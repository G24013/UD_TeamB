1. チェックポイント．（ピリオド）とは
チェックポイント．（ピリオド）は、フリーフォントです。
和文総合書体処女作「チェックポイントフォント」をベースとしながらも、
手書きフォント処女作「ぎゃーてーるみねっせんす」の
細い横画要素も盛り込み、15×15ドットのビットマップフォント風
アウトラインフォントへと、さらなる進化を遂げました。

イメージ元の『アメリカ横断ウルトラクイズ』からは
かなりかけ離れたデザインとなりましたが、
便宜上、クイズフォントの一つとして位置づけしています。

チェックポイントフォントシリーズの最終作にして、
弊サイトの集大成的作品です。
作者のフォントにかける情熱をこれでもかと注ぎ込みましたので、
そのようなことは特にお気になさらずお使いいただければと思います。



2. 収録文字
・ひらがな
・カタカナ…全角/半角
・アルファベット…全角/半角
・数字…全角/半角
・記号…ギリシア文字、キリル文字、その他一式網羅。アクサンにも対応
・漢字…JIS第一/第二水準6,355字＋IBM拡張文字
	＋日本漢字能力検定1級配当漢字（一部を除く）＋α　計：約6,900字



3. 動作環境
現在のところ、
Microsoft Windows XP、Vista、7、8、10、11で動作を確認しております。

商用利用（同人誌などでの使用）についても、

ご自由にお使いいただければ制作者冥利に尽きます。

使用許可連絡も必要ございませんが、

web拍手やXなどでご一報いただければありがたいです。

なお、

6. その他
当フォントは、『アメリカ横断ウルトラクイズ』の二次創作物です。

収録されている文字・記号の種類および字形、文字幅、変換割り当て、
またダウンロード用のリンクなどは予告なく変更する場合があります。
