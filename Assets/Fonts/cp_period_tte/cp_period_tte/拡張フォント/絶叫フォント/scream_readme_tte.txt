1. 絶叫チェックポイントピリオドとは
絶叫チェックポイントピリオドは、
拡張チェックポイントピリオド（圧縮ファイル同封）の字形を
さらに拡張するフォントです。

収録文字はひらがな・カタカナのみですが、
捨て仮名を含む全ての文字に濁点が付いております。
絶叫しているような独特の雰囲気を醸し出せるかもしれません。

メインフォントと併せてインストールしていただき、
必要に応じてご活用いただければと思います。



2. 収録文字
・ひらがな…濁点付き
・カタカナ…全角のみ（濁点付き）
・アルファベット…なし
・数字…なし
・記号…なし
・漢字…「絶」「叫」の2字のみ



3. 動作環境
現在のところ、
Microsoft Windows XP、Vista、7、8、10、11で動作を確認しております。

商用利用（同人誌などでの使用）についても、
特に制限は設けておりません。
ご自由にお使いいただければ制作者冥利に尽きます。

使用許可連絡も必要ございませんが、
よろしければ事後でも結構ですので、
web拍手やXなどでご一報いただければありがたいです。

なお、
